# 查看视频信息

macOS Finder 右键快速操作，一秒查看视频分辨率、帧率、编码、时长。

## 功能

- 📐 分辨率（自动识别横屏/竖屏）
- 🎞 帧率
- 🎬 视频编码
- ⏱ 时长
- 💾 文件大小

## 安装

1. 下载 [查看视频信息.pkg](../../releases/latest)
2. 双击安装，输入管理员密码
3. 右键任意视频文件 → 快速操作 → 查看视频信息

## 支持格式

MP4、MOV、MKV、AVI 等主流格式。

## 系统要求

- macOS 13 或更高版本
- Apple Silicon（M 系列芯片）

## 卸载

```bash
sudo rm -rf "/Library/Services/查看视频信息.workflow"
```
